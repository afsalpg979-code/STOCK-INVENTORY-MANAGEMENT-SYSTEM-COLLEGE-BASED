Steps to Run Inventory Management System Using XAMPP
1. Install XAMPP Server
Download and install XAMPP on your system.

Open XAMPP Control Panel.

2. Move Project Folder
Copy your project folder (e.g., inventory).

Paste it into the htdocs directory:

C:\xampp\htdocs\
3. Start Server Services
Open XAMPP Control Panel.

Click Start for:

Apache

MySQL

4. Create Database
Open browser and go to:

http://localhost/phpmyadmin
Create a new database named:

inventory
5. Import SQL File
Select the inventory database.

Click on Import tab.

Choose the file:

inventory.sql (UPDATED SQL File)
Click Import.

6. Run the Project
Open browser and type:

http://localhost/foldername
(Replace “foldername” with your project folder name)
